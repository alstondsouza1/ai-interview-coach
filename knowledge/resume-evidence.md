---
title: Evidence-Based Resume Project Bullets
source: https://careers.microsoft.com/v2/global/en/hiring-tips.html
---

A useful project bullet connects an action, technical method, and outcome.
Name the tool only when it supports a real action. Add scale through users,
tests, performance, time saved, team size, or another honest measurement.

Tailoring should improve relevance without inventing experience. A missing
job skill belongs in a study plan unless the candidate can truthfully connect
it to completed work.
