# Three-Minute Hackathon Demo

## 0:00-0:30 - Problem

Students receive generic interview advice and unexplained match percentages.
Interview Prep Studio makes every recommendation traceable.

## 0:30-1:05 - Evidence-Backed Analysis

Load the fictional profile, build the workspace, and show:

- Required versus preferred qualifications
- Missing required skills
- Exact resume excerpts supporting matched skills

## 1:05-1:40 - Mock Interview

Open Mock Interview and show:

- 10 balanced questions
- One-question-at-a-time flow
- Adjustable timer
- Transparent rubric and final category charts

## 1:40-2:15 - Foundry IQ

Open Grounded Coach. Ask how to prepare for a missing skill. Enable Foundry IQ,
confirm consent, and show:

- Cited answer
- Source viewer
- Retrieval activity

## 2:15-2:45 - Progress

Open Progress and show score trends. Explain that SQLite stores aggregate
scores only, not resumes or answers.

## 2:45-3:00 - Close

> Interview Prep Studio combines private, explainable assessment with
> consent-based, citation-backed Microsoft intelligence.
