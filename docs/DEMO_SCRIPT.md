# Three-Minute Demo Video Script

Target length: **3:00**. Record at 1280×720 or higher. Use the fictional sample
profile only — no real resume, employer, or personal data on screen. Speak the
**Say** lines; perform the **Show** actions.

Before recording: start the app (`streamlit run app.py`), confirm Foundry IQ is
configured (sidebar shows "Foundry IQ configured"), and have the Streamlit page
navigation visible.

---

## 0:00–0:25 — Problem & hook

**Say:** "Students get generic interview advice and unexplained match scores.
Interview Prep Studio makes every recommendation traceable — and keeps personal
data local."

**Show:** Title / landing of the app.

## 0:25–1:00 — Evidence-backed role fit

**Show:** Sidebar → **Load example profile** → **Build my preparation
workspace**. Open the **Overview** tab.

**Say:** "It separates required from preferred qualifications, flags missing
skills, and — most importantly — links every matched skill to the exact resume
line that earned it."

**Show:** Expand one matched skill to reveal its resume excerpt.

## 1:00–1:35 — Timed mock interview

**Show:** Open **Mock Interview** from the page navigation. Pick a timer, start,
answer one question, submit.

**Say:** "A balanced ten-question interview — four technical, three behavioral,
three situational — one at a time, with a countdown and a fully visible scoring
rubric. No hidden model decides the score."

**Show:** Briefly show the category/rubric charts on completion (or a pre-run
result).

## 1:35–2:20 — Grounded Coach with Foundry IQ

**Show:** Open **Grounded Coach**. Toggle **Use Microsoft Foundry IQ**, check
consent, ask: *"How should a student structure a teamwork answer for a software
engineering internship?"*

**Say:** "Grounded Coach calls a Microsoft Foundry IQ knowledge base. The answer
comes back with citations and a retrieval-activity panel — endpoint, knowledge
base, elapsed time, and citation count — so guidance is traceable to a source.
Only the reviewed question is sent; the resume never leaves the device."

**Show:** The green "Live retrieval from Microsoft Foundry IQ" banner, a citation
expander, and the retrieval-activity metrics.

## 2:20–2:45 — Progress & privacy

**Show:** Open **Progress**; show the readiness trend.

**Say:** "Progress stores aggregate scores only in local SQLite — never resume or
answer text — and you can delete all history in one click."

## 2:45–3:00 — Close

**Say:** "Interview Prep Studio combines private, explainable assessment with
consent-based, citation-backed Microsoft Foundry IQ intelligence. Built with
GitHub Copilot. Thanks for watching."

**Show:** Repository URL / live demo URL on screen.

---

## Recording tips

- Do a dry run once; the Foundry call takes a few seconds — keep talking through
  it or trim in editing.
- If the network is unreliable during recording, the local cited fallback still
  demonstrates citations; mention it is the offline fallback.
- Keep total runtime at or under 3:00 — practice the transitions.
