# Privacy Notice

Interview Prep Studio is designed around data minimization.

## Processed Locally

- Resume text
- Job description text
- Skill evidence
- Practice answers
- Mock-interview answers
- Deterministic rubric scores

These values remain in the active Streamlit session and are not written to the
history database.

## Stored Locally

The SQLite history stores only:

- Date and session identifier
- Target role title
- Overall readiness score
- Category and rubric averages
- Average response time
- Aggregate strengths and weaknesses

Users can delete all stored history from the Progress page.

## Optional Foundry IQ Processing

Foundry IQ is disabled unless configured. A user must enable it and confirm
consent before each request. The application sends only the text shown in the
coaching question box.

The application does not automatically send:

- Full resume text
- Job description text
- Practice answers
- Mock-interview answers

Cloud processing is governed by the Azure subscription and Microsoft product
terms configured by the project owner.

## Demo Data

All repository sample data is fictional. Do not use real customer, employer,
or confidential Microsoft information in screenshots or demonstrations.
