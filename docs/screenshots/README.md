# Screenshots

Add the images referenced by the project README and the GitHub Copilot
development journal here. Keep every image fictional — no real resume,
employer, or personal data.

## Needed for submission

| File name | What to capture |
| --- | --- |
| `role-fit.png` | Main workspace showing required vs. preferred skills and an expanded evidence match |
| `mock-interview.png` | A timed mock-interview question with the countdown visible |
| `grounded-coach.png` | A Foundry IQ (or local fallback) answer with its citation trail |
| `progress.png` | The Progress page score trend chart |
| `copilot-chat.png` | GitHub Copilot Chat assisting during development |
| `copilot-suggestion.png` | An accepted or modified Copilot code suggestion |

Capture on macOS with `Cmd+Shift+5`. Save the files with the exact names
above so the README image links resolve.
