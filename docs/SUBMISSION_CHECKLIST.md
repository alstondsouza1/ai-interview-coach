# Hackathon Submission Checklist

## Challenge Requirements

- [ ] Project linked to **Creative Apps**
- [ ] Foundry IQ resource created and connected
- [ ] Knowledge base populated with attributed content
- [ ] Live Grounded Coach response includes citations
- [ ] Real GitHub Copilot usage documented
- [ ] Copilot screenshots added without confidential information

## Security

- [ ] `.streamlit/secrets.toml` is not tracked
- [ ] No API keys, tokens, passwords, or connection strings in Git history
- [ ] GitHub push protection enabled
- [ ] Sample data is fictional
- [ ] `python scripts/check_public_repo.py` passes

## Product Quality

- [ ] `pytest -q` passes
- [ ] All Streamlit pages open successfully
- [ ] Mobile layout reviewed
- [ ] Keyboard navigation reviewed
- [ ] Foundry fallback tested
- [ ] Progress history can be deleted

## Submission Assets

- [ ] Public repository URL
- [ ] Live deployment URL
- [ ] Three-minute demo video
- [ ] Architecture screenshot
- [ ] Foundry IQ knowledge-base screenshot
- [ ] Foundry citation screenshot
- [ ] GitHub Copilot usage screenshots
- [ ] Project description and challenge selection
